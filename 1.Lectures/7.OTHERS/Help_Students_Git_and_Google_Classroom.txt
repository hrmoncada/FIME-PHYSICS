================================
Git
================================
1. Instalar Git
Descárguelo desde:
https://git-scm.com/

2. Abrir una terminal (CMD)
Puede usar Terminal, Consola, o el Símbolo del sistema (CMD).

3. Navegar hasta el directorio donde desea descargar el repositorio de Mecánica de Fluidos
$ cd Desktop        # Ir al escritorio
$ dir               # Listar archivos (en Windows)
$ mkdir Guapp       # Crear una carpeta llamada Guapp
$ cd Guap

4. Clonar el repositorio desde GitHub
$ git clone https://github.com/hrmoncada/UNAC-FIME-FLUID-MECHANICS

5. Actualizar su repositorio local

Primero, ingrese al directorio del repositorio descargado:
$ cd UNAC-FIME-FLUID-MECHANICS

Verifique si hay cambios pendientes en su repositorio local:
$ git status

Ejecute la actualización:
$ git pull

6. Celular o móvil
También puede instalar Git en su teléfono para monitorear las novedades del curso.
Disponible en Google Play Store.
